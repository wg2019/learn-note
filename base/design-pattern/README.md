# 设计模式 

---
## 模型分类
* **创建型模式**：五种。[*工厂方法模式*][1]、[*抽象工厂模式*][2]、[*单例模式*][3]、[*建造者模式*][4]、[*原型模式*][5]。
* **结构性模式**：七种。[*适配器模式*][6]、[*装饰者模式*][7]、[*代理模式*][8]、[*外观模式*][9]、[*桥接模式*][10]、[*组合模式*][11]、[*享元模式*][12]。
* **行为型模式**：十一种。[*策略模式*][13]、[*模板方法模式*][14]、[*观察者模式*][15]、[*迭代子模式*][16]、责任链模式、命令模式、备忘录模式、状态模式、访问者模式、中介者模式、解释器模式。
* **并发型模式**、**线程池模式**。
